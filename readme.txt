Take payments via MoneyUnify on your WooCommerce store.

== Description ==

This plugin allows you to integrate MoneyUnify as a payment gateway on your WooCommerce store. Customers can make payments using their mobile money accounts.

= Features =

- Secure payment processing with MoneyUnify.
- Seamless integration with WooCommerce.
- Option to send SMS notifications to admin and customers for new orders and order completion.
- Customizable sender ID and admin phone number for SMS notifications.

== Installation ==

1. Upload the `moneyunify-woocommerce` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to WooCommerce > Settings > Payments and configure the MoneyUnify payment gateway.
4. Optionally, configure the SMS notification settings in the plugin settings.

== Frequently Asked Questions ==

= Can I use this plugin without WooCommerce? =

No, this plugin is specifically designed to work with WooCommerce.

= How do I configure SMS notifications? =

After activating the plugin, go to WooCommerce > Settings > Payments > MoneyUnify, and you'll find options to enable SMS notifications and configure sender ID, admin phone number, and ExciteSMS API key.
